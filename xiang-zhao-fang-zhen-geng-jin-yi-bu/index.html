向着仿真更近一步

· 2020-07-04 · # 专业知识 # 上升阶段仿真

开始着手准备编写数值积分模块和微分方程模块，理论上讲应该是现成的，但是比较繁琐，还是一步一步来的好。

一、数值积分模块
数值积分算法采用四阶龙格库塔法，在这里正好复习一下。数值积分算法中，常见的有：欧拉法、改进欧拉法、龙格库塔法和adams法。以微分方程x˙=f(x)\dot x =f(x)x˙=f(x)为例
(1)欧拉法
假设步长为hhh，f(x)f(x)f(x)表示右端函数值（就是微分方程），那么递推公式可以表示为
xn+1=xn+h∗f(xn)x_{n+1} = x_n + h*f(x_n)xn+1​=xn​+h∗f(xn​)
别看这个公式简单，其实是很常见的。在构建扰动观测器得到积分变量时，就是用的这种方法。
(2)改进欧拉法
也称为梯形公式，采用平均的思想来计算所谓的增量，递推公式表示如下：
xn+1=xn+h2∗[f(xn)+f(xn+1)]x_{n+1} = x_n + \dfrac{h}{2}*[f(x_n)+f(x_{n+1})]xn+1​=xn​+2h​∗[f(xn​)+f(xn+1​)]
同时也需要注意到，上式其实是无法应用的，计算xn+1x_{n+1}xn+1​需要用到f(xn+1)f(x_{n+1})f(xn+1​)，而计算f(xn+1)f(x_{n+1})f(xn+1​)又需要用到xn+1x_{n+1}xn+1​。这是一个鸡生蛋，蛋生鸡的问题，解不了。
所以，又提出了一种所谓的预估校正法。
先预估一次
xˉn+1=xn+h∗f(xn)\bar x_{n+1} = x_n + h*f(x_n)xˉn+1​=xn​+h∗f(xn​)
然后再校正
xn+1=xn+h2∗[f(xn)+f(xˉn+1)]x_{n+1} = x_n + \dfrac{h}{2}*[f(x_n)+f(\bar x_{n+1})]xn+1​=xn​+2h​∗[f(xn​)+f(xˉn+1​)]
代入，并写成标准形式
xn+1=xn+h2∗[f(xn)+f(xn+h∗f(xn))]x_{n+1} = x_n + \dfrac{h}{2}*[f(x_n)+f(x_n + h*f(x_n))]xn+1​=xn​+2h​∗[f(xn​)+f(xn​+h∗f(xn​))]
(3)龙格库塔法
如果说前面的改进欧拉法，体现了一种平均的思想，这里的龙格库塔法就是平均思想的进一步延续：加权平均。可以总结为三步走：半步+半步+一步
第一步
k1=f(xn)k_1 = f(x_n)k1​=f(xn​)
预估一次
xˉn+11=xn+h2∗k1\bar x_{n+1}^1=x_n+\dfrac{h}{2}*k_1xˉn+11​=xn​+2h​∗k1​

第二步
k2=f(xˉn+11)k_2 = f(\bar x_{n+1}^1)k2​=f(xˉn+11​)
预估一次
xˉn+12=xn+h2∗k2\bar x_{n+1}^2=x_n+\dfrac{h}{2}*k_2xˉn+12​=xn​+2h​∗k2​

第三步
k3=f(xˉn+12)k_3 = f(\bar x_{n+1}^2)k3​=f(xˉn+12​)
预估一次
xˉn+13=xn+h∗k3\bar x_{n+1}^3=x_n+h*k_3xˉn+13​=xn​+h∗k3​

k4=f(xˉn+13)k_4 = f(\bar x_{n+1}^3)k4​=f(xˉn+13​)

最后，采用加权平均的形式来表示
xn+1=xn+h2∗(k1+2k2+2k3+k4)x_{n+1} = x_n + \dfrac{h}{2}*(k1+2k_2+2k_3+k_4)xn+1​=xn​+2h​∗(k1+2k2​+2k3​+k4​)
(4)Adams法
这种方法常用于时实计算领域，类型和四阶龙格库塔算法相似，采用的是预估校正的思想。形式上稍微复杂一些，不再深入讨论。
这里主要是复习了常见的数值积分算法，在程序中将采用四阶龙格库塔算法来实现数值积分。
