六自由度模型是可以完整描述平流层飞艇的上升过程的。哪六个自由度呢，从本体坐标系来看，三个运动自由度和三个转动自由度。根据平流层飞艇的运动特点，滚转可以忽略不计，这样可以适当简化问题的分析：三个运动量和两个转动量（俯仰和航向）。三个运动量是必须要有的，这样可以描述飞艇上升的轨迹。俯仰运动是需要描述的，因为净浮心变化主要影响的就是俯仰运动。航向运动也很重要，但不是那么重要，飞艇不同的航向，会影响飞行轨迹。如果考虑风场的影响话，不同的飞行姿态，飞艇受到的气动力是不同的，进而也会影响其运动。
