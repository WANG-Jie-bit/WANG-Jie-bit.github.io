霍曼转移：航天器的变轨问题

· 2020-07-24 ·

昨天，中国火星探测器“天问”终于上天，据说需要历经7个月的时间才能到达火星。探测器从地面发射升空到达火星究竟是一个怎样的过程？让我突然想起来之前课上学习过的一个内容——霍曼转移。

到底什么是霍曼转移呢？霍曼转移是在同一个平面内，从一个轨道转移到另一个轨道最节省燃料的方式。

初始轨道是一个圆轨道，目标轨道也是一个圆轨道，转移轨道是与这两个圆轨道相切的椭圆形轨道。对于转移到火星上的过程就更加复杂，因为探测器不仅仅要转移到火星轨道上，还需要转移到轨道上时正好被火星捕获。

1、霍曼转移的基本问题
这里先不讨论火星捕获的问题，先简单说一下霍曼转移的基本问题。先引入一个能量的概念，比机械能。
ϵ=V22−μR=−μ2a\epsilon = \dfrac{V^2}{2} - \dfrac{\mu}{R}=-\dfrac{\mu}{2a}ϵ=2V2​−Rμ​=−2aμ​
其中，aaa指的是轨道半径，对于椭圆轨道来说，指的是半长轴。

那么，在各轨道（初始轨道、转移轨道和目标轨道）上，能量的大小就可以表示为：
ϵ1=−μ2R1\epsilon _1 = -\dfrac{\mu}{2R_1}ϵ1​=−2R1​μ​
ϵ2=−μ2R2\epsilon _2 = -\dfrac{\mu}{2R_2}ϵ2​=−2R2​μ​
ϵ转移=−μ(R1+R2)\epsilon _{转移} = -\dfrac{\mu}{(R_1+R_2)}ϵ转移​=−(R1​+R2​)μ​

下面简单讨论变轨的过程。当航天器运行至1时，增大速度，此时离心力大于地球的万有引力，航天器不再保持圆形轨道绕地球转动，而是变成了椭圆轨道。这个过程为1次变轨。当航天器运行至2时，继续增大速度，变轨至大圆轨道，这个过程为2次变轨。根据比机械能，可以简单计算出两次变轨所需要的速度增量。

在小圆轨道上，速度可以计算为：
V1=2(ϵ1+μR1)V_1 = \sqrt{2(\epsilon _1 + \dfrac{\mu}{R_1})}V1​=2(ϵ1​+R1​μ​)​
在大圆轨道上，速度可以计算为：
V2=2(ϵ2+μR2)V_2 = \sqrt{2(\epsilon _2 + \dfrac{\mu}{R_2})}V2​=2(ϵ2​+R2​μ​)​
在椭圆轨道上1点，速度可以计算为：
V1∗=2(ϵ转移+μR1)V_1^* = \sqrt{2(\epsilon _{转移} + \dfrac{\mu}{R_1})}V1∗​=2(ϵ转移​+R1​μ​)​

2、非共面变轨问题
主要考虑的是两个不在同一平面内的轨道变轨问题。

首先考虑的是一个简单的变轨问题：假设两个轨道形状一样，只是不在一个平面内。此时，变轨时不需要改变速度大小，只需要考虑速度方向的改变问题。

4、交会问题
探测器由地球出发，最终被火星捕获，这个问题本质上就是一个变轨问题，同时需要考虑交会问题。为了保证探测器能够被火星捕获，要求探测器在变轨结束之后正好和火星相遇。

变轨转移时间在上面的分析中已经求得，要使得火星和探测器能够在目标轨道上相遇，需要求火星在tft_ftf​时间内能够到达交会点。火星与交会点之间的角度为α=ω2tf\alpha=\omega _2 t_fα=ω2​tf​。
定义火星-太阳连线与探测器-太阳联系之间的角度为相位角ϕ\phiϕ，当ϕ=π−α\phi=\pi-\alphaϕ=π−α时，满足变轨条件。此时改变探测器的速度，可以实现与火星的交会。

但是，如果不满足上述条件呢？
有这样一种现象：在低轨道的探测器，角速度是大于高轨道的火星的，也就是说探测器转的快。这样，还是有机会满足上述条件的。这个问题本质上就是小学数学中的追击问题。假设角速度差是Δω\Delta \omegaΔω，初始相位角是ϕ0\phi _0ϕ0​目标相位角是ϕf\phi _fϕf​，那么需要的时间为
Δt=ΔϕΔω\Delta t = \dfrac{\Delta \phi}{\Delta \omega}Δt=ΔωΔϕ​

这里需要注意Δϕ\Delta \phiΔϕ的定义问题。若初始相位角较大，表明探测器位置位于目标点位置之后，则Δϕ=ϕ0−ϕf\Delta \phi=\phi _0 - \phi _fΔϕ=ϕ0​−ϕf​；如初始相位角较小，表明探测器位置位于目标点位置之前，则Δϕ=ϕ0−ϕf+2π\Delta \phi=\phi _0 - \phi _f + 2 \piΔϕ=ϕ0​−ϕf​+2π

当然，上述讨论都是基于理想条件给出的初步计算结果。实际情况中，火星轨道和探测器初始轨道可能都是椭圆，而且不在同一个平面，这样的话问题就要复杂一些，需要建立更为精确的模型来进行描述。
